# 💊 MediMate – Medicine Scanner App 📸
A user-friendly mobile app that allows users to **scan medicine labels** using their device's camera for quick and efficient **medicine identification**. The app features a **simplistic design** with a smooth user experience, making it easy for anyone to use.

## 🚀 Features
| Feature                | Description                                          |
|------------------------|------------------------------------------------------|
| ✅ Medicine Scanner     | Capture and scan medicine labels for instant identification. |
| ✅ Image Processing     | Crop and adjust images for better recognition.       |
| ✅ Smooth & Simple UI   | A clean and intuitive interface for seamless navigation. |
| ✅ Fast & Efficient     | Quick medicine recognition and real-time responses.  |

## 🛠️ Services
| Service                | Description                                          |
|------------------------|------------------------------------------------------|
| ✅ AI Doctor (Gemini API) | Built-in AI assistant that **only** discusses health-related topics. |
| ✅ Nearby Hospitals      | Locate nearby hospitals with distance, directions, and contact details. |
| ✅ Emergency Contacts    | Quick access to emergency services and personal SOS alerts. |
| ✅ Consult a Doctor      | Book appointments, chat, or video call with certified doctors. |

## 📰 News Articles
| News Feature          | Description                                          |
|------------------------|------------------------------------------------------|
| ✅ Daily News Feed      | Get real-time updates on global health news.         |
| ✅ Medicine Alerts      | Receive notifications on medicine recalls and safety warnings. |
| ✅ Health Tips & Articles | Read expert articles on wellness, medical research, and healthy living. |
| ✅ Trending Topics      | Stay updated with the latest trends and breakthroughs in healthcare. |

## 🔗 Installation & Usage
| Step                | Command                                              |
|---------------------|------------------------------------------------------|
| 1. Clone Repository  | `git clone https://github.com/yourusername/medimate.git` |
| 2. Install Dependencies | `npm install` or `yarn install`                      |
| 3. Run on Android    | `npx react-native run-android`                        |
| 4. Run on iOS        | `npx react-native run-ios`                            |

## 🤖 Tech Stack
| Technology                      | Purpose                                         |
|----------------------------------|-------------------------------------------------|
| **React Native**                | Mobile app development                         |
| **React Native Image Crop Picker** | Image selection & cropping                      |
| **Gemini API**                  | AI Doctor with health-specific responses       |
| **Google Maps API**             | Locating nearby hospitals                      |
| **Geolocation API**             | Emergency and hospital location services      |
| **News API**                    | Fetching real-time health and medical news    |
